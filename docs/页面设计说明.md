# 前端页面设计说明

## 设计系统

基于 ui-ux-pro-max 设计系统生成的专业设计规范：

- **风格**: Accessible & Ethical（高对比度、可访问性）
- **字体**: EB Garamond（标题）+ Lato（正文）
- **配色**: 专业蓝色系（Primary: #0F172A, CTA: #0369A1）
- **原则**: WCAG AAA 可访问性标准

## 已实现的页面

### 1. 登录页面 (`LoginView.vue`)
- 简洁专业的登录表单
- 用户名/密码输入
- 记住我功能
- 表单验证
- 响应式设计

### 2. 主布局 (`AppLayout.vue`)
- 顶部导航栏（Logo、消息通知、用户信息）
- 侧边栏导航菜单（可折叠）
- 主内容区域
- 路由过渡动画

### 3. 工作台 (`DashboardView.vue`)
- 统计卡片（档案总数、已数字化、待处理任务、今日新增）
- 最近访问的档案列表
- 快捷操作按钮
- 系统公告时间线

### 4. 档案管理 (`ArchiveListView.vue`)
- 搜索和筛选功能
- 档案列表表格
- 分页功能
- 操作按钮（查看、下载、删除）

### 5. 档案详情 (`ArchiveDetailView.vue`)
- 基本信息展示
- 分类信息
- 标签展示
- OCR识别文本
- 元数据展示

### 6. 档案导入 (`ArchiveImportView.vue`)
- 拖拽上传文件
- 批量上传支持（最多1000个文件）
- 上传进度显示
- 导入任务列表

### 7. 文档生成 (`DocGenerateView.vue`)
- 文档类型选择
- 模板选择
- 核心要素输入
- 关联档案选择
- 生成任务列表

### 8. 知识图谱 (`KnowledgeGraphView.vue`)
- 查询条件面板
- 实体类型选择
- 关联关系查询
- 图谱可视化区域（待集成 ECharts/D3.js）

### 9. 统计分析 (`StatisticsView.vue`)
- 多维度筛选
- 统计图表展示（待集成 ECharts）
- 数据表格
- 报表导出功能

## 路由配置

所有路由已配置在 `router/index.ts`：
- `/login` - 登录页
- `/dashboard` - 工作台
- `/archive` - 档案列表
- `/archive/import` - 档案导入
- `/archive/:id` - 档案详情
- `/doc-generate` - 文档生成
- `/knowledge-graph` - 知识图谱
- `/statistics` - 统计分析

路由守卫已实现：
- 登录状态检查

## 使用说明

### 开发环境启动

```bash
cd frontend
pnpm install
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

### 代码检查

```bash
pnpm lint
```

## 注意事项

1. **可访问性**: 所有交互元素都添加了 `cursor-pointer`，确保良好的用户体验
2. **响应式**: 所有页面都支持移动端、平板和桌面端
3. **加载状态**: 所有异步操作都显示加载状态
4. **错误处理**: 统一的错误提示和处理机制
5. **代码规范**: 遵循 Vue 3 Composition API 最佳实践
